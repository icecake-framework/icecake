Button Snippet

Example of use

Embeddable snippet

tag<ick-button ... />

example: <ick-button Title="Home" IsOutlined HRef='/index.html'/>

Properties

HRef: specifies the url link associated with the button. HRef is optional, if it's defined the button type is automatically set to BTN_TYPE_A and renders as a <a> element.

Title: the text rendered inside the button. Can be a simple text or a more complex html string.

IsOutlined: Outlined button style.

IsRounded: Rounded button style.

IsLoading: Loading button state.

Style

The ickecake ui.button is based on the bulma css button, so any additional bulma class can be used.
