Image Snippet

Example of use

Embeddable snippet

tag<ick-image ... />

example: <ick-image Size="is-96x96" Src="/icecake.svg"/>

Properties

Src: the url for the source of the image

Size: the size or the ratio of the image

Style

IsRounded: Rounded image style

The ickecake ui.image is based on the bulma css image, so any additional bulma class can be used.
